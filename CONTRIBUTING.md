# How to contribute to Itomori

_Thanks for decided to contribute to Itomori_ 

## Did you find a bug?

If the bug is crashing the app, please feel free to make a quick **pull request.** 
If the bug is not crashing the app but maybe causing probelms, you can create a **issue** or make a **pull request** .

## Did you find any spelling or grammer mistakes?

You can create a **pull request** or a issue with a **documentation** tag.

## Do you want to add some new features?

First, discuss about that at **Discussions** of Itomori in Github. If you recive more positive feedbacks, feel free to make a **pull request**.


**Thanks for using Itomori ❤️**
